 A CRUD for form in Node.js n mySQL.


## Installation
*for newbies : Clone or download zip to your machine then hit this :

	npm install
	
	
## Running
- `node app.js`
- Running @`localhost:4300`
- Navigate to `/customers`
- If `/customers` nit working then go to `/customers/add`
